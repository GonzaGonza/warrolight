warrolight (server)
===================

This is the WarroLight server component. It contains a websocket server for adjusting and simulating lights.

It also contains the code for all the lights programs and the configuration required to make them run.

A configuration is:
 * A geometry module
 * A shape mapping module. Gives name to different shapes
 * Number of lights
 * Output devices (UDP o Serial)
 * lightsToDevicesMapping
